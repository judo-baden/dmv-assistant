# DMV Assistant (Tampermonkey)

Der **DMV Assistant** ist ein Tampermonkey-Userscript zur **Automatisierung** und **Vereinfachung** von Abläufen im **DJB‑Portal** (`https://djb.dokume.app/`).

Das Projekt ist vollständig in **TypeScript** umgesetzt und wird mit **esbuild** in eine einzige `*.user.js`‑Datei gebündelt.

## Hauptfunktionen

- **CSV‑Import mit Fortschrittsanzeige**: Ein-Klick‑Import von Sammelbestell‑Datensätzen, Fortschrittsbalken, Dark/Light‑Theme, robustes SPA-Handling
- **Automatische Bereinigung von Lizenzbestellungen**: Löscht alle problematischen Positionen, automatische Systempopup‑Bejahung, Fortschrittsanzeige
